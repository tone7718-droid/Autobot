# 🩺 내 통증 사용설명서

동네 병원(정형외과·재활의학과)에서 가장 자주 진단되는 **근골격계 질환 30가지**를
재활 트레이닝·스포츠의학 관점에서 정리한 웹 전자책입니다.
고등학생 수준의 독해력이면 충분히 이해할 수 있는 쉬운 말로 풀었습니다.

## 구성

7개 부위 카테고리 × 30개 질환, 모든 질환은 같은 구조로 정리되어 있습니다.

| 섹션 | 내용 |
|---|---|
| 📖 어떤 질환인가요? | 질환의 정체와 원인을 비유로 쉽게 설명 |
| ✅ 증상 체크 | 내 증상과 비교하는 체크리스트 |
| 🔬 자가 평가 | 임상 검사를 집에서 안전하게 해보는 방법 |
| 🤲 수동 치료 | 도수치료·물리치료 등 전문가가 해주는 치료 |
| 🏃 능동 치료 | 횟수·세트까지 제시된 단계별 셀프 운동 |
| 🚨 병원에 가야 할 때 | 자가 관리를 멈춰야 하는 위험 신호(red flag) |

### 수록 질환 (30)

- **목 (3)** — 거북목/일자목, 목 디스크, 근막통증증후군(담 결림)
- **어깨 (5)** — 날개뼈 운동이상증, 어깨충돌증후군, 회전근개 건병증/파열, 오십견, 석회성 건염
- **팔꿈치·손목·손 (5)** — 테니스 엘보, 골프 엘보, 손목터널증후군, 드퀘르벵 건초염, 방아쇠 수지
- **허리·골반 (5)** — 비특이적 요통, 허리 디스크, 척추관 협착증, 천장관절 기능장애, 이상근 증후군
- **엉덩이·허벅지 (3)** — 고관절 충돌증후군(FAI), 대전자 통증 증후군, 햄스트링 부상
- **무릎 (4)** — 슬개대퇴 통증 증후군(러너스 니), 슬개건염(점퍼스 니), 장경인대 증후군, 무릎 골관절염
- **발목·발 (5)** — 발목 염좌/만성 불안정증, 족저근막염, 아킬레스 건병증, 정강이 통증(신 스플린트), 무지외반증

## 기술 구조

빌드 과정이 전혀 없는 **순수 정적 사이트**(HTML + CSS + Vanilla JS)입니다.

```
index.html          # 단일 페이지 앱 진입점
css/style.css       # 전체 스타일
js/app.js           # 해시 라우터 + 렌더러 + 증상 검색
js/data/*.js        # 카테고리별 질환 콘텐츠 (7개 파일, 30개 질환)
```

- 해시 라우팅: `#/category/neck`, `#/condition/tennis-elbow`, `#/search/검색어`
- 증상 키워드 검색: 질환명·증상·원인 텍스트 전체에서 매칭
- 서버·DB·빌드 도구 불필요 — 정적 호스팅이면 어디서든 동작

## Vercel 배포

1. [vercel.com](https://vercel.com)에서 **Add New → Project**
2. 이 GitHub 저장소를 Import
3. Framework Preset: **Other**, Build/Output 설정은 모두 기본값(비워 둠)
4. **Deploy** — 끝. 빌드 단계가 없으므로 바로 배포됩니다.

로컬 미리보기:

```bash
npx serve .        # 또는
python3 -m http.server 8000
```

## ⚠️ 면책 안내

이 전자책은 건강 정보 제공을 목적으로 하며, 의료인의 진단과 치료를 대신할 수 없습니다.
각 질환의 "병원에 가야 할 때" 항목에 해당하거나 증상이 2주 이상 지속되면 의료기관을 방문하세요.
